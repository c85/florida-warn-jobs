# FloridaJobs WARN Notice Alerts

## Description:
This is an Azure Python function that sends an email alert whenever a [WARN notice](https://floridajobs.org/office-directory/division-of-workforce-services/workforce-programs/reemployment-and-emergency-assistance-coordination-team-react/warn-notices) is posted in the state of Florida (this can be modified within the code to fit your needs). For more information on the WARN Act, [click here](https://floridajobs.org/office-directory/division-of-workforce-services/workforce-programs/worker-adjustment-and-retraining-notification-(warn)-act).
